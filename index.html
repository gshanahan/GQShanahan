Projects

PocketProverbs

PocketProverbs is a GPT-powered Bible study assistant built to enhance Scripture exploration with AI-driven insights. Using the OpenAI GPT API, the app delivers historical context, verse analysis, and cross-references. The frontend is developed with HTML and JavaScript, while Google Firebase manages user authentication and stores study data. Hosted on Namecheap, this full-stack web application provides a seamless experience, streamlining Bible study by offering curated, data-driven resources and insights.
